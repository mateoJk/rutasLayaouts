# GameVault

## Descripción

GameVault es una aplicación web para explorar, buscar y gestionar información sobre videojuegos. Permite a los usuarios navegar por un catálogo de juegos, ver detalles de cada título y organizar su colección personal.

## Getting Started

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

Abrí [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.
